🌐 Web Scraper Using BeautifulSoup

📌 Project Overview

This project is a simple web scraper built using Python and BeautifulSoup. It extracts the latest news headlines and their corresponding links from the Hacker News website and displays them in the terminal.

The project demonstrates the basics of web scraping, HTML parsing, and data extraction using Python.

🚀 Features

- Fetches live webpage content from Hacker News
- Extracts news headlines
- Extracts article links
- Displays results in a structured format
- Beginner-friendly and easy to understand

🛠️ Technologies Used

- Python
- Requests
- BeautifulSoup4

📂 Project Structure

web-scraper-python/

├── web_scraper.py

├── README.md

└── headlines.txt (optional)

⚙️ Installation

Install the required libraries:

pip install requests beautifulsoup4

▶️ How to Run

Run the Python file:

python web_scraper.py
